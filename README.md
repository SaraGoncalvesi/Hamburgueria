# i9lar – Site Institucional

## Descrição do Projeto

Este projeto foi desenvolvido como uma aplicação web institucional para a empresa i9lar, com o objetivo de apresentar os serviços oferecidos, facilitar o contato com clientes e fortalecer a presença digital da empresa.

O site foi construído com foco em design moderno, responsividade e organização das informações, proporcionando uma experiência intuitiva para o usuário.

O projeto também foi desenvolvido como uma avaliação prática para o curso bacharelado Sistemas de Informação da UNIP.

---

## Objetivo

O principal objetivo do projeto é servir como vitrine digital da empresa, permitindo que clientes conheçam os serviços, entrem em contato e tenham acesso rápido às informações institucionais.

---

## Funcionalidades

- Página inicial com apresentação da empresa
- Seção de serviços oferecidos
- Formulário de orçamento
- Página de contato com informações e links diretos
- Integração com redes sociais
- Layout responsivo para dispositivos móveis
- Navegação simples e intuitiva entre páginas

---

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Bootstrap 
- Design responsivo

## Desenvolvimento

O projeto foi desenvolvido com foco em:
